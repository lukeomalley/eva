> Eva is a dialect of lisp implemented in JavaScript. This project was build with the purpose of understanding how an interpreter for a dialect of lisp works.

## Examples

Simple Arithmetic

```
['+', ['*', 3, 2], 5] => 6

['-', ['-', ['*', 5, 5], 2], 3] => 20
```

Variables

```
['var', 'x', 5]

'x' => 5
```
